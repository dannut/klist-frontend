info

kli.st is a knowledge base engineered for DevOps professionals and System Administrators. In complex containerized and orchestrated environments, memorizing every syntax nuance for tools like Kubernetes, Docker, or Linux is highly inefficient.
kli.st bridges this gap by delivering instant, full-text search across more than 5000 CLI commands.

features

  • Full-text search across 5000+ commands
  • 20 DevOps tools covered
  • One-click copy to clipboard
  • CLI tool for terminal use (kli search docker)
  • Open source

covered tools

Docker Kubernetes Git Terraform AWS CLI Azure CLI GCloud Helm ArgoCD Ansible Jenkins Linux Prometheus Grafana Vault Redis Kafka Harbor Nginx Flux

get kli

Use kli.st directly from your terminal. No root required, no dependencies.

terminal 
install:
$ curl -sL https://test.kli.st/install.sh | bash
usage:
$ kli search docker
$ kli search "list containers"
$ kli search tool=kubernetes
$ kli --help
